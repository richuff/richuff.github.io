2026.04.15 前端

Query 是一款快速、轻量且跨浏览器兼容的 JavaScript 库，核心理念为 Write Less, Do More（写更少代码，做更多事）。它以 CSS 选择器为基础，封装了 DOM 操作、事件处理、动画效果与 Ajax 交互等常用功能，提供简洁统一的 API，大幅降低原生 JS 编写成本与浏览器兼容负担。自 2006 年发布以来，它凭借链式调用、插件生态与高效开发体验广泛普及，至今仍在大量传统 Web 项目与快速原型开发中发挥作用。
